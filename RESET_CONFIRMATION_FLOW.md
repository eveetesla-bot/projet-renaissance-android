# Parcours de confirmation des réinitialisations

## Recommencer la séance

1. L’utilisateur ouvre le panneau « Recommencer la séance ».
2. Le texte précise que seule la session ouverte est concernée.
3. Pendant l’opération, le bouton affiche « Redémarrage… » et ne peut pas être
   pressé deux fois.
4. Après succès, le panneau se ferme, l’exercice 1 réapparaît, les séries, RPE,
   note, douleur et repos sont remis à zéro.
5. Un message de réussite peut être fermé manuellement.
6. En cas d’erreur, aucune remise à zéro visuelle n’est appliquée.

## Journée, profil et total

Le centre « Remise à zéro » se trouve dans Profil. Chaque bouton ouvre un texte
spécifique avant d’afficher « Confirmer » :

- journée : historique antérieur et santé conservés ;
- profil : données locales du profil actif supprimées ;
- total : première installation, autorisations santé révoquées, données
  Health Connect conservées sur le téléphone.

L’action n’est lancée qu’après confirmation explicite.
